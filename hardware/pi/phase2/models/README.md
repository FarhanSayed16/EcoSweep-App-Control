# Phase 2 models folder

Place here:

- **detect.tflite** — Your TFLite object-detection model (e.g. SSD MobileNet, EfficientDet-Lite).
- **labels.txt** — One class name per line, in the same order as the model’s class indices (e.g. COCO labels).

See `docs/phase-2-ai/PHASE-2-FULL-GUIDE.md` for where to get models and how to run detection.
